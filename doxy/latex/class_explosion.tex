\doxysubsection{Detailed Description}
This class represents an explosion. 

This clas inherits from sf\+::\+Sprite. The animation of an explosion is a frame by frame animation of 4 sucessive images. It has a contructor \doxylink{class_explosion_a3686b2a9c2072b43726734bdba034b81}{Explosion\+::\+Explosion()} that starts the explosion when called. The method \doxylink{class_explosion_a19f6595805de83b1a6ee894b88fbee09}{Explosion\+::update()} that switches to the next frame.
