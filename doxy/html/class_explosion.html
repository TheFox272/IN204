Detailed Description

This class represents an explosion.

This clas inherits from sf::Sprite. The animation of an explosion is a frame by frame animation of 4 sucessive images. It has a contructor Explosion::Explosion() that starts the explosion when called. The method Explosion::update() that switches to the next frame.
